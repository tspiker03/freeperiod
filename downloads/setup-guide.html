FreePeriod Pro Toolkit

Setup Guide — You'll be up and running in about 15 minutes.

What You Got

Your Pro Toolkit includes:

  • 3 MCP Bundles (.mcpb files) — Canvas LMS (15 tools), Google Workspace (13 tools), Microsoft 365 (14 tools)
  • System Prompt — Custom instructions with 9 workflow commands
  • This Setup Guide

Prerequisites

  1. Claude Pro subscription ($20/month from Anthropic) — claude.ai/settings/billing
  2. Claude Desktop app (free) — claude.ai/download
Important: The .mcpb bundles require the Claude Desktop app, not the web version. Make sure you have Claude Desktop installed before proceeding.

Step 1: Install the MCP Bundles

Install the bundles for the platforms your school uses. You don't need all three.

Canvas LMS

  1. Double-click freeperiod-canvas.mcpb
  2. Claude Desktop will open and ask you to confirm the installation
  3. Click Install
  4. When prompted, enter your Canvas API token (get one from your Canvas Settings > Approved Integrations > New Access Token)
  5. Enter your Canvas URL (e.g., https://yourschool.instructure.com)

Google Workspace

  1. Double-click freeperiod-google.mcpb
  2. Click Install in Claude Desktop
  3. A browser window will open for Google OAuth — sign in with your school Google account
  4. Grant the requested permissions (Classroom, Gmail, Calendar, Drive, Docs, Sheets, Slides, Forms)
  5. The browser will show "Authentication successful" — you can close it

Microsoft 365

  1. Double-click freeperiod-microsoft.mcpb
  2. Click Install in Claude Desktop
  3. You'll see a device code — go to microsoft.com/devicelogin and enter it
  4. Sign in with your school Microsoft account
  5. Grant the requested permissions
Tip: You can install just the bundle(s) for your school's platform. If your school uses Canvas and Google, install those two. You can always add more later.

Step 2: Set Up the System Prompt

  1. Open Claude Desktop
  2. Click Projects in the sidebar
  3. Click Create project
  4. Name it "FreePeriod" (or whatever you like)
  5. Click Set custom instructions
  6. Open the system-prompt.txt file from your download
  7. Copy the entire contents and paste into the Custom Instructions field
  8. Click Save

Step 3: Try It Out

Start a new conversation inside your FreePeriod project and try these commands:

/plan photosynthesis 7th grade science — Generate a lesson plan

/rubric persuasive essay 10th grade — Create a rubric

/quiz American Revolution 8th grade, 10 questions — Build a quiz

/grade — Pull assignments from your LMS and help grade them

/parent-email Jordan missing work — Draft a parent email

/sub-plans — Generate sub plans from your current unit

All 9 Commands

  • /plan — Standards-aligned lesson plans
  • /grade — Rubric-based grading assistance
  • /parent-email — Professional parent communication
  • /quiz — Assessments with answer keys
  • /sub-plans — Emergency substitute teacher plans
  • /differentiate — Three-tier lesson adaptation
  • /rubric — Assessment rubric generator
  • /report — Student/class performance summaries
  • /standards — Standards lookup and alignment

Troubleshooting

MCP bundle won't install

Make sure you have the latest version of Claude Desktop. The .mcpb format requires a recent update.

Canvas says "unauthorized"

Your API token may have expired. Generate a new one in Canvas: Settings > Approved Integrations > New Access Token.

Google auth fails

Make sure you're signing in with the same Google account you use for Google Classroom. If your school uses Google Workspace for Education, you may need your admin to allow third-party apps.
