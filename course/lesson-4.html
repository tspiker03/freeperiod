Lesson 4

Assessment Building with /quiz

Create quizzes, tests, and exit tickets with mixed question types, answer keys, and standards alignment. Plus: what's next.

⏱ About 15 minutes

📝 Why /quiz is more than a quiz maker

The /quiz command creates assessments with pedagogical rigor built in. Every quiz it generates includes:

  • Mixed question types — multiple choice, short answer, and higher-order thinking
  • Standards alignment — each question tied to a specific standard
  • Answer key — with point values and acceptable answer variations
  • Bloom's taxonomy coverage — not just recall questions

You can use it for formal tests, quick exit tickets, formative checks, or even study guides.

🎯 Activity: Build Different Types of Assessments

1

Create a quiz for an upcoming unit

/quiz American Revolution, 8th grade US History, 15 questions, include a document-based question

Notice how Claude includes questions at different cognitive levels — not just "who" and "when" but "why" and "what if."

2

Make a quick exit ticket

/quiz solving two-step equations, 7th grade math, 3 questions only, focus on common misconceptions

Great for a 5-minute check at the end of class.

3

Build a study guide

"Turn that quiz into a study guide with the questions but not the answers. Add a vocabulary review section."
4

Create multiple versions

"Create a Version B of this quiz with the same standards but different questions, so I can use it for makeups"

💡 Advanced Assessment Techniques

  • Specify DOK levels. "Include at least 3 DOK level 3 questions" for more rigorous assessments.
  • Target misconceptions. "Create distractors based on common student mistakes with fractions" makes multiple choice more diagnostic.
  • Combine with /differentiate. "Create a modified version of this quiz for my approaching-level students with word banks and fewer questions."
  • Assessment for learning. "After each question, add a note about what skill it assesses, so I can identify which students need reteaching on which skills."

🏆 Putting It All Together

Now that you know all 4 commands, try this complete workflow for an upcoming unit:

1
/plan [your unit topic] [grade]

Generate the lesson plan.

2
/differentiate the lesson above

Adapt it for all learners.

3
/rubric [the main assignment from the lesson]

Create the rubric for assessment.

4
/quiz [the standards from the lesson] as an exit ticket, 5 questions

Build a formative assessment to close the lesson.
